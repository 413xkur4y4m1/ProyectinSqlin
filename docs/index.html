Sistema de Préstamos ULSA

Objetivo del Proyecto

El Sistema de Préstamos ULSA es una plataforma web diseñada para gestionar eficientemente el préstamo de equipos y materiales en la Universidad La Salle. Facilita el proceso tanto para alumnos como para administradores, proporcionando una interfaz intuitiva y segura.

Características Principales

  • Autenticación segura con Microsoft y correo institucional
  • Gestión de préstamos en tiempo real
  • Sistema de seguimiento de adeudos
  • Panel de administración intuitivo
  • Integración con Firebase para tiempo real

Metodología SCRUM

Sprint 1 (2 semanas)

- Configuración inicial del proyecto

- Implementación de autenticación básica

- Diseño de la interfaz de usuario principal

Sprint 2 (2 semanas)

- Integración con Microsoft Azure AD

- Implementación de FingerprintJS

- Sistema de roles y permisos

Sprint 3 (2 semanas)

- Sistema de préstamos y devoluciones

- Gestión de adeudos

- Notificaciones en tiempo real

Arquitectura del Sistema

Frontend

- HTML5, CSS3, JavaScript - Bootstrap 4.5.0 - Firebase SDK 8.10.1 - FingerprintJS v3

Backend

- Firebase Authentication - Firebase Realtime Database - Node.js Server (Express) - Microsoft Azure AD Integration

Flujo de Autenticación

1. Usuario → Login Form
2. Auth System → Microsoft/Email
3. Firebase Auth → Verify
4. FingerprintJS → Device Check
5. Database → User Profile

Seguridad del Sistema

Autenticación Microsoft

El sistema implementa dos métodos de autenticación con Microsoft:

  • OAuth 2.0 con Azure AD
  • Firebase Authentication

FingerprintJS

Implementación de identificación de dispositivos para:

  • Prevención de fraude
  • Detección de inicio de sesión sospechoso
  • Verificación de dispositivos autorizados
